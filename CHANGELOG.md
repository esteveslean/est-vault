# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-03-09

### Added

- Initial release of est-vault
- Core vault encryption/decryption with AES-256-GCM
- CLI commands: `create`, `view`, `edit`, `encrypt`, `decrypt`, `rekey`, `version`
- Non-interactive usage for CI/CD pipelines
- Comprehensive test suite with 43 tests
- GitHub Actions workflows for testing, linting, and releases
